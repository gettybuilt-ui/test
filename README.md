# 21st.dev Magic MCP Test

This project is configured with the [21st.dev Magic MCP](https://github.com/21st-dev/magic-mcp) server for AI-driven UI component generation.

## Prerequisites

- Node.js 18+

## Setup

1. Get an API key from https://console.21st.dev/magic
2. When prompted by your IDE, enter the API key

The MCP server configuration is in `.mcp.json`.

## Usage

Use the `/ui` command in your IDE chat to generate UI components from natural language descriptions. For example:

```
/ui a modern login form with email and password fields
```
